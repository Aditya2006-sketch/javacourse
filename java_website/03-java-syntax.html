Java Syntax

Main Method

The main() method is essential in every Java application. It tells the Java Virtual Machine (JVM) where the program starts. This method must be declared as public static void and must accept a String[] as its parameter.

public static void main(String[] args) {
  // code to execute
}

Explanation:

  • public - accessible from anywhere
  • static - does not require an object to run
  • void - does not return any value
  • main - method name where execution begins
  • String[] args - command-line arguments

System.out.println()

System.out.println() is used to print a line of text to the console. Each statement ends with a semicolon (;), and the text to print is enclosed in double quotes.

System.out.println("Hello Java");

Explanation:

  • System - a built-in class that provides access to system resources
  • out - a static member (PrintStream) of the System class
  • println() - prints a message followed by a new line

Complete Example

This is a complete Java program demonstrating the use of the main method and System.out.println().

public class Main {
  public static void main(String[] args) {
    System.out.println("Hello Java");
    System.out.println("Learning Java is fun!");
  }
}
