Java Arrays

Arrays in Java are used to store multiple values in a single variable, instead of declaring separate variables for each value. They are fixed in size and hold elements of the same data type.

Declaring an Array

To declare an array in Java: 

int[] myArray;  // Preferred
// or
int myArray[];

Creating an Array

Once declared, you can create an array using the new keyword: 

myArray = new int[5];

Initializing an Array

You can initialize an array at the time of declaration: 

int[] numbers = {10, 20, 30, 40, 50};

Accessing Array Elements

Array elements are accessed using index numbers starting from 0: 

System.out.println(numbers[0]);  // Outputs 10

Example Program

public class ArrayExample {
  public static void main(String[] args) {
    int[] arr = {1, 2, 3, 4, 5};

    for (int i = 0; i < arr.length; i++) {
      System.out.println("Element at index " + i + ": " + arr[i]);
    }
  }
}
