Java Output

In Java, we use System.out.println() to display output on the console. It helps us show messages, results, values of variables, and more while the program runs. Java also supports printing text, numbers, and special characters using escape sequences like \n (newline) or \" (double quotes).

Printing Text

To display text in Java, we use System.out.println(). It prints the given string and moves the cursor to the next line.

System.out.println("Hello, world!");

Printing Numbers

You can directly print numbers using System.out.println() without quotes.

System.out.println(123);
System.out.println(3.14);

Printing Multiple Values

You can also print text and numbers together using + to concatenate:

System.out.println("Age: " + 20);
System.out.println("Sum: " + (5 + 3));

Printing Double Quotes

To print double quotes inside a string, use the backslash (\) escape character before each quote.

System.out.println("She said, \"Java is awesome!\"");

Complete Example

Here's a complete Java program demonstrating different output types:

public class OutputExample {
  public static void main(String[] args) {
    System.out.println("Java Output Demo");
    System.out.println(42);
    System.out.println("Temperature: " + 36.6);
    System.out.println("Total: " + (10 + 5));
    System.out.println("She said, \"Java is awesome!\"");
  }
}

Output:

Java Output Demo
42
Temperature: 36.6
Total: 15
She said, "Java is awesome!"
