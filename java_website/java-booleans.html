Java Boolean

A boolean expression returns either true or false. It is commonly used in conditions and logic operations.

Boolean Expression

You can use comparison operators to check if a condition is true or false.

int x = 10;
int y = 9;
System.out.println(x > y); // returns true

Boolean Values

Java has a boolean data type that can store only two values: true or false.

boolean isJavaFun = true;
boolean isTasty = false;

System.out.println(isJavaFun);
System.out.println(isTasty);

Boolean Expressions in Conditional Testing

Boolean expressions are often used in conditional statements like if or while to make decisions.

int x = 10;

System.out.println(x == 15); // false
System.out.println(x == 10); // true

Note: Booleans are the basis of all Java comparisons and conditional logic.
