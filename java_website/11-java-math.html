Java Math Methods

Java provides many methods in the Math class to perform mathematical tasks on variables.

Finding Maximum and Minimum

Math.max(x, y) returns the highest value of x and y:

Math.max(5, 10);

Math.min(x, y) returns the lowest value of x and y:

Math.min(5, 10);

Square Root

Math.sqrt(x) returns the square root of x:

Math.sqrt(64);

Absolute Value

Math.abs(x) returns the absolute (positive) value of x:

Math.abs(-6.7);

Random Numbers

Math.random() returns a random number between 0.0 (inclusive) and 1.0 (exclusive):

Math.random();

To get more control over the random number, use this formula:

int randomNum = (int)(Math.random() * 101); // 0 to 100
