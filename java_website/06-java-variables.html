What are Variables?

In Java, variables are containers for storing data values. Java is a statically typed language, so you must declare a variable's type before using it.

Declaring Variables

int number = 10;
String name = "Alice";
Here, number is an integer, and name is a string.

Identifiers

An identifier is the name used for variables, methods, classes, etc. It must follow specific rules:

  • Can contain letters, digits, underscore, and dollar sign
  • Must begin with a letter, underscore (_), or dollar sign ($)
  • Cannot begin with a digit
  • Cannot be a reserved keyword (like class, int)
int age = 25;      // valid
String _name = "Bob"; // valid
int 2ndNumber = 50; // invalid

Variable Types

  • int - stores integers
  • double - stores decimal numbers
  • char - stores a single character
  • String - stores text
  • boolean - stores true or false

Types of Variables

Java supports three types of variables based on their scope and declaration:

  • Local Variables - Declared inside a method or block and accessible only there
  • Instance Variables - Declared in a class, but outside any method. They belong to objects.
  • Static Variables - Declared using the static keyword and shared among all instances of a class
class Student {
  int id; // instance variable
  static String college = "ABC College"; // static variable

  void display() {
    String name = "John"; // local variable
    System.out.println(name + " " + id + " " + college);
  }
}

Multiple Variables

int x = 5, y = 10, z = 15;
You can declare and assign multiple variables in a single line.

Final Variables

final int SPEED = 90;
A variable declared with final is a constant and cannot be changed.

Output Example

String language = "Java";
System.out.println("I am learning " + language);
Output: I am learning Java
