Java Switch Statement

The switch statement in Java allows you to execute different blocks of code based on the value of a variable.

Syntax:

switch (expression) {
  case value1:
    // code block
    break;
  case value2:
    // code block
    break;
  ...
  default:
    // default code block
}

Example:

int day = 4;
switch (day) {
  case 1:
    System.out.println("Monday");
    break;
  case 2:
    System.out.println("Tuesday");
    break;
  case 3:
    System.out.println("Wednesday");
    break;
  case 4:
    System.out.println("Thursday");
    break;
  default:
    System.out.println("Weekend");
}

Key Points:

  • switch evaluates the expression once and compares it to each case.
  • If a match is found, the corresponding block runs.
  • break stops the switch block from falling through.
  • default executes if no match is found.
