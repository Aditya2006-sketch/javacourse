Java While Loop

A while loop in Java is a control flow statement that allows code to be executed repeatedly based on a given Boolean condition. The loop continues to execute as long as the condition evaluates to true.

Syntax

while (condition) {
    // Code to be executed
}

Key Points

  • Condition Check: The condition is evaluated before the loop body is executed.
  • Infinite Loop: If the condition never becomes false, the loop will run indefinitely.
  • Use Case: Ideal for scenarios where the number of iterations is not predetermined.

Example

public class WhileLoopExample {
    public static void main(String[] args) {
        int count = 1;

        // While loop to print numbers from 1 to 5
        while (count <= 5) {
            System.out.println("Count is: " + count);
            count++; // Increment the counter
        }
    }
}

Output

Count is: 1
Count is: 2
Count is: 3
Count is: 4
Count is: 5
