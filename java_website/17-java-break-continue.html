Java break and continue Statements

In Java, the break and continue statements are used to control the flow of loops. They allow you to either exit a loop early or skip certain iterations based on a condition.

Break Statement

  • Exits the loop completely when a certain condition is met.
  • Commonly used inside loops or switch statements.
  • Only exits the current loop, not nested ones (unless labeled).
// Example of break
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break;
    }
    System.out.println(i);
}

Continue Statement

  • Skips the current iteration and jumps to the next one.
  • Does not exit the loop.
  • Useful for skipping specific values or steps.
// Example of continue
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        continue;
    }
    System.out.println(i);
}
