Java Strings

Numbers and Strings

1. Adding Numbers — the result will be a number:

int x = 10;
int y = 20;
int z = x + y;

2. Adding Strings — the result will be a string (concatenation):

String x = "10";
String y = "20";
String z = x + y;

3. Adding a Number and a String — the result will be a string (concatenation):

String x = "10";
int y = 20;
String z = x + y;

Creating a String

A string can be created using double quotes and assigned to a variable.

String name = "John";

String Length

You can find the number of characters in a string using the length() method.

System.out.println(name.length());

Finding a Character in a String

Use the charAt(index) method to find the character at a specific position (index starts at 0).

String name = "Blackboard";
System.out.println(name.charAt(5));

Finding Index of a Character

You can use the indexOf(char) method to find the position of a character.

System.out.println(name.indexOf('a'));

More String Methods

  • toUpperCase() - Convert to uppercase
  • toLowerCase() - Convert to lowercase
String name = "Blackboard";
System.out.println(name.toUpperCase());
System.out.println(name.toLowerCase());

String Concatenation

You can combine strings using + or the concat() method:

String firstName = "John";
String lastName = "Doe";
System.out.println(firstName + " " + lastName);
System.out.println(firstName.concat(lastName));
