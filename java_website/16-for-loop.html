Java For Loop

The for loop in Java is used when you know how many times you want to execute a block of code.

Syntax:

for (initialization; condition; update) {
  // code block
}

Example:

for (int i = 0; i < 5; i++) {
  System.out.println(i);
}

Key Points:

  • Initialization is executed once at the beginning.
  • Condition is evaluated before each loop iteration.
  • Update is executed after each loop iteration.
